# Security domains

## Sensitive security context fields

* `runAsUser` shoudn't be equal to 0 (root)

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: security-context-demo
spec:
  securityContext:
    runAsUser: 0
  containers:
    - name: sec-ctx-demo
      image: gcr.io/google-samples/node-hello:1.0
```

* `allowPrivilegeEscalation` should stay at false

```yaml
apiVersion: v1
kind: Pod
metadata:
  name: security-context-demo
spec:
  securityContext:
    runAsUser: 1000
  containers:
    - name: sec-ctx-demo-2
        image: gcr.io/google-samples/node-hello:1.0
        securityContext:
        runAsUser: 2000
        allowPrivilegeEscalation: true
```

* `readOnlyRootFilesystem` should stay at false

```yaml
apiVersion: v1  
kind: Pod  
metadata:  
  name: hello-world  
spec:  
  containers:  
  securityContext:  
    readOnlyRootFilesystem: false
    runAsNonRoot: true
```
